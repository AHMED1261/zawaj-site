# موقع الجواز - دليل الاستخدام (النسخة الجديدة)

## هيكل النظام
- تسجيل حساب: رقم هاتف + كلمة مرور (بدون OTP حاليًا)
- بعد التسجيل: استمارة كاملة (57 سؤال) إلزامية بالكامل، بمنطق ذكي (الأسئلة اللي مالهاش داعي بتتخفي تلقائيًا)
- بعد الإرسال: الحساب يبقى "قيد المراجعة" ومايظهرش في البحث
- المشرف بيراجع من `admin.html` ويوافق/يرفض
- بعد الموافقة بس، البروفايل يظهر في `browse.html` مع الفلاتر

## الصفحات
| الصفحة | الوظيفة |
|---|---|
| `register.html` | إنشاء حساب (هاتف + باسورد + نوع) |
| `login.html` | تسجيل الدخول |
| `form.html` | استمارة الـ 57 سؤال (بعد تسجيل الدخول) |
| `browse.html` | تصفح البروفايلات المعتمدة + فلاتر (كتابة واختيار) |
| `admin-login.html` | دخول المشرف |
| `admin.html` | مراجعة الحسابات + توليد بيانات تجريبية |

## متغيرات البيئة الجديدة المطلوبة
بالإضافة لبيانات Aiven القديمة، ضيف في Environment Variables على Hostinger:
```
JWT_SECRET=نص عشوائي طويل وسري (استخدمه لتوقيع جلسات الدخول)
ADMIN_USERNAME=admin
ADMIN_PASSWORD=باسورد قوي تختاره
```
حساب المشرف بيتعمل تلقائيًا أول ما السيرفر يشتغل، من غير ما تحتاج SQL يدوي.

## اختبار الفلاتر بـ 200 بروفايل تجريبي
1. سجّل دخول على `admin.html` بحساب المشرف
2. اضغط "توليد 200 بروفايل تجريبي"
3. روح `browse.html` وجرّب الفلاتر مباشرة

## ملحوظات مهمة
- التسجيل حاليًا بدون OTP فعلي (مباشر بالهاتف والباسورد) — لو عايز تضيف تحقق حقيقي بالهاتف لاحقًا، هيحتاج اشتراك SMS مدفوع.
- كل الأسئلة إلزامية إلا اللي بتتحدد بإجابة سابقة (زي أسئلة الطلاق اللي بتظهر بس لو الحالة "مطلق").
- الرقم القومي وصورة البطاقة (المتفق عليهم في مستند المواصفات) لسه مش متضافين في الكود ده — دي خطوة تالية.
