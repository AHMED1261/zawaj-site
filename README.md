# موقع الجواز - دليل النشر على Hostinger

## 1) تجهيز قاعدة البيانات
1. ادخل hPanel -> **Databases -> MySQL Databases**
2. اعمل داتابيز جديدة (مثلاً `zawajdb`) ويوزر جديد بباسورد قوي
3. سجّل القيم دي: `DB_HOST`, `DB_USER`, `DB_PASSWORD`, `DB_NAME`
   (عادة `DB_HOST` بيكون `localhost` لو التطبيق والداتابيز على نفس الحساب)

## 2) رفع الكود
عندك طريقتين من hPanel -> **Websites -> Add Website -> Node.js App**:

**أ) عن طريق GitHub (الأفضل):**
- ارفع الفولدر ده على مستودع GitHub خاص بيك
- اربط الحساب في Hostinger بـ GitHub واختار الريبو
- Hostinger هيكتشف إنه Express تلقائيًا وينفذ `npm install` و `npm start`

**ب) رفع ZIP مباشر:**
- اعمل ضغط للفولدر (من غير `node_modules`)
- ارفعه من نفس الصفحة بخيار File Upload

## 3) متغيرات البيئة
في إعدادات التطبيق على hPanel هتلاقي قسم **Environment Variables**، ضيف فيه:
```
DB_HOST=localhost
DB_USER=القيمة اللي سجلتها
DB_PASSWORD=القيمة اللي سجلتها
DB_NAME=القيمة اللي سجلتها
PORT=3000
```
(ملف `.env.example` المرفق هو مرجع بس - متسيبوش يترفع بالباسورد الحقيقي)

## 4) التشغيل
- Hostinger هيشغل `npm start` تلقائيًا وهيربط الدومين/الساب دومين بالتطبيق
- أول تشغيل هيعمل الجدول (`profiles`) تلقائيًا لوحده (الكود فيه `initSchema()`)

## 5) الاختبار محليًا قبل الرفع (اختياري)
```bash
cp .env.example .env
# عدّل القيم بتاعتك جوه .env
npm install
npm start
```
افتح `http://localhost:3000/register.html` و `http://localhost:3000/browse.html`

## الفلاتر المدعومة حاليًا
- السن (من - لحد)
- النوع (ذكر/أنثى)
- الحالة الاجتماعية (أعزب/مطلق/أرمل)
- عايز يخلف؟ (نعم/لا/لسه مقررش)
- المحافظة

## إضافات مستقبلية سهلة
- رفع صورة البروفايل (Hostinger بيدي مساحة تخزين، أو استخدم رابط خارجي)
- تسجيل دخول وصفحة "بروفايلي" للتعديل
- نظام رسائل بين المستخدمين
- توثيق الإيميل بكود تفعيل
