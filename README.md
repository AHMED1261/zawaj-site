# منصة الزواج

منصة تعارف جادة باللغة العربية، تعمل بالمراجعة الإدارية ولا تعرض صورًا أو وسائل تواصل مباشرة بين المستخدمين.

## ما أُضيف في هذه النسخة

- تأكيد البريد الإلكتروني برمز قبل إنشاء الحساب.
- استرجاع كلمة المرور برمز يصل إلى البريد الإلكتروني المرتبط بالحساب.
- إشعارات بريدية للحساب المعتمد/المرفوض، طلبات التعارف، الموافقة، ورسائل الإدارة.
- لوحة مشرف تتضمن مراجعة الحسابات وإدارة المراسلات بين المتوافقين.
- تحسين تجربة الواجهة على الموبايل وخط عربي وحالات تركيز أوضح.

## التشغيل

1. انسخ `.env.example` إلى ملف باسم `.env` وأكمل بيانات قاعدة البيانات.
2. ضع قيمة عشوائية قوية بطول 32 حرفًا على الأقل في `JWT_SECRET`.
3. نفّذ `npm install` ثم `npm start`.

## إعداد البريد الإلكتروني المجاني

الموقع يستخدم Resend لإرسال البريد. الخطة المجانية الحالية مناسبة للبداية وتسمح بـ 3,000 رسالة شهرياً، بحد 100 رسالة يومياً. ستحتاج إلى:

1. حساب Resend مجاني ومفتاح API.
2. نطاق أو عنوان مرسل تم التحقق منه.
3. ضع `RESEND_API_KEY` و`EMAIL_FROM` في `.env`.

لن يتم إرسال أو إظهار أي رمز في المتصفح. إذا كانت إعدادات البريد ناقصة، يوقف الموقع التسجيل بالرمز بدل تقديم تحقق غير آمن.
